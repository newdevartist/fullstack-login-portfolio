# Backend - Flask Authentication API

This is the backend of the fullstack-login-portfolio project, built with Flask.

## Features
- User Registration
- User Login
- Password hashing using Werkzeug

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/fullstack-login-portfolio.git
    cd fullstack-login-portfolio/backend
    ```

2. Create a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use venv\Scripts\activate
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the Flask server:
    ```bash
    python main.py
    ```

The backend will be running on `http://127.0.0.1:5000`.
