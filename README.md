# Fullstack Login Portfolio

Αυτό είναι ένα πλήρες έργο Fullstack που περιλαμβάνει μια εφαρμογή **Login** με **React** στο frontend και **Python (Flask/Django)** στο backend. Το έργο επιτρέπει την εγγραφή και σύνδεση χρηστών, καθώς και την εμφάνιση μιας dashboard με τα δεδομένα του χρήστη μετά την σύνδεση.

## Τεχνολογίες

- **Frontend**: React, JavaScript, CSS
- **Backend**: Python, Flask (ή Django, ανάλογα με το τι χρησιμοποίησες)
- **Βάση Δεδομένων**: MongoDB (ή MySQL/PostgreSQL αν χρησιμοποίησες SQL)
- **Authentication**: JWT (JSON Web Tokens)

## Χαρακτηριστικά

- Δημιουργία χρήστη (sign-up)
- Σύνδεση χρήστη (login)
- Διαχείριση συνεδρίας με JWT
- Δυνατότητα εμφάνισης Dashboard με προσωπικά δεδομένα του χρήστη
- Responsive σχεδίαση
